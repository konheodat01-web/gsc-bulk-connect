Hướng dẫn cài đặt
Cách cấu hình Google OAuth để sử dụng tool
Bước 1 — Tạo Google OAuth Client ID
1. Vào Google Cloud Console → tạo Project mới
2. APIs & Services → Enable APIs → bật Google Search Console API
3. Credentials → Create Credentials → OAuth 2.0 Client ID → Web application
4. Authorized JavaScript origins: thêm URL GitHub Pages
5. Authorized redirect URIs: thêm URL GitHub Pages
6. Copy Client ID → paste vào tab Cấu hình
Bước 2 — OAuth Consent Screen
1. APIs & Services → OAuth consent screen → External → Create
2. Điền tên app, email → Next
3. Scopes → Add scope → tìm webmasters → thêm cả 2 scope
4. Test users → thêm email Google của bạn (chế độ Testing)
5. Quay lại tool → đăng nhập → dùng được!
